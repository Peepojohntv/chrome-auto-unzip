## GERMAN:

### Beschreibung
Mit dieser Chrome Extension kannst du heruntergeladene `.zip`-Dateien automatisch entpacken.  
Der Inhalt wird direkt im Standard-Download-Ordner gespeichert – ganz ohne manuelles Eingreifen.

**Funktionen:**
- Erkennt automatisch, wenn eine `.zip`-Datei heruntergeladen wurde  
- Entpackt den Inhalt automatisch im Download-Ordner  
- Funktioniert vollständig im Hintergrund  
- Keine zusätzliche Software nötig  

### Installation
1. Chrome öffnen → `chrome://extensions/`  
2. Rechts oben **„Entwicklermodus“** aktivieren  
3. **„Entpackte Erweiterung laden“** auswählen  
4. Den lokal heruntergeladenen Ordner der Erweiterung auswählen  
5. Fertig! Ab sofort werden neue `.zip`-Dateien automatisch entpackt.

## ENGLISH:

### Description
With this Chrome extension, you can automatically extract downloaded `.zip` files.  
The extracted contents are saved directly in your default Downloads folder — no manual work required.

**Features:**
- Detects automatically when a `.zip` file is downloaded  
- Extracts its contents in the Downloads folder  
- Runs silently in the background  
- No additional software required  

### Installation
1. Open Chrome → `chrome://extensions/`  
2. Activate **"Developer mode"** in the top right corner  
3. Click **"Load unpacked extension"**  
4. Select the downloaded extension folder  
5. Done! New `.zip` files will now be automatically extracted.****
